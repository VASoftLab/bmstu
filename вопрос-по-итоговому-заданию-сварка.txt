В задаче «Прогнозирование размеров сварного шва при электронно-лучевой сварке тонкостенных конструкций аэрокосмического назначения» в нашем распоряжении ограниченный набор данных. При этом, часть датасета - это размеры шва для одинаковой комбинации параметров (IW, IF, VW, FP) - по 4 значения целевых переменных для каждой уникальной комбинации входных параметров. Возникла идея - получить синтетический обучающий датасет, путем группировки исходного набора по уникальным комбинациям параметров (IW, IF, VW, FP) и сопоставления им усредненных значений целевых переменных для каждой группы из 4-х строк. В результате получился датасет из 15 строк. А данные из исходного набора - 72 строки - использовать для тестирования. Тогда разбиение исходного датасета (72 строки) на обучающую и тестовую не требуется.
